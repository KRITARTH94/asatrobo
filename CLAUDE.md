# Project Instructions

## Git workflow

After making any code change in this repo, automatically commit and push it to `origin/main` on GitHub (https://github.com/KRITARTH94/asatrobo) — do not wait to be asked. Use a short, descriptive commit message summarizing the change.

Git is installed at `C:\Program Files\Git` but is not on PATH by default in this environment; add it for the session before running git commands:

```powershell
$env:Path += ";C:\Program Files\Git\cmd"
```
